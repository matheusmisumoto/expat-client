EXPAT - Sistema colaborativo de custo de vida nas cidades

Sobre o EXPAT

A proposta deste site é auxiliar você no planejamento de finanças pessoais ao migrar de cidade, seja por motivos profissionais ou pessoais. Os dados são abastecidos por usuários voluntários, que realizam um cadastro prévio no site.

O cadastro é necessário para evitar distorções nos dados. Nenhuma informação pessoal é repassada a terceiros.

EXPAT é um projeto desenvolvido para a disciplina "Desenvolvimento para Servidores II", do curso de Sistemas para Internet na Faculdade de Tecnologia do Estado de São Paulo (FATEC), campus Baixada Santista.

Este site utiliza Java, Java Server Faces, Spring Boot, MySQL, BootStrap.
